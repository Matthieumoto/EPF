# 📌 EPF - Cours de Python 🐍  

Bienvenue dans ce dépôt contenant mes codes réalisés dans le cadre de l'UE **Python** de mon cursus à l'EPF.  

## 📂 Contenu du dépôt  
Ce dossier regroupe tous les scripts, exercices et projets que je développe durant les cours de programmation en Python.  

### 📌 Objectif du module  
L'UE Python à l'EPF vise à :  
✅ Apprendre les bases du langage Python (syntaxe, structures de contrôle, fonctions...)  
✅ Développer des compétences en algorithmique et en résolution de problèmes  
✅ Manipuler des structures de données (listes, dictionnaires, tuples...)  
✅ Travailler avec des bibliothèques populaires (NumPy, Pandas, Matplotlib...)  
✅ Automatiser des tâches et analyser des données  

### 📁 Structure du dépôt  
📜 **/Exercices/** → Exercices pratiques et corrigés  
📜 **/Projets/** → Projets réalisés au cours de l'année  
📜 **/TP/** → Travaux pratiques guidés  
📜 **/Notes/** → Notes et ressources utiles  

## 🚀 Comment utiliser ce dépôt ?  
1️⃣ Cloner le dépôt en local :  
   ```bash
   git clone https://github.com/ton-utilisateur/EPF-Python.git
   cd EPF-Python
   ```  
2️⃣ Exécuter un fichier Python :  
   ```bash
   python nom_du_fichier.py
   ```  
3️⃣ Installer les dépendances si nécessaire :  
   ```bash
   pip install -r requirements.txt
   ```
